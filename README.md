# bingo-generator
A lightweight Python project that creates a local web interface for generating bingo cards and randomly drawing bingo numbers.

## Features

- Generate bingo cards
- Random bingo number generator
- Runs locally in your browser
- No internet connection required

## Requirements

- Python 3.x
